About Me

Profile

I am a front-end web developer with desire to build user-friendly websites and applications. My chief goal is always to create awesome applications that clients love to use for their services.

Development & Design

Dealing with the front-end part of the web excites me. I find it enjoyable working on projects involving web User Interface as well as User Experience. I have worked with front-end technologies including CSS3, HTML5, vanilla Javascript, React, git and often use netlify as my development deployment platform.

Time in development

I believe in shipping quality code. Therefore, I put much effort on improving my coding skills; working round the clock on both client and side projects, implementing best coding practices, as well as learning new technology stack. I vividly know the value of time, hardwork and honesty in relation to delivering quality work.

Learning and Training

I pursued Forensic Science in Kenyatta University for my Bachelor's degree. Back then, doing class projects in C++, made my interest for coding level up. I decided to make use of available online resources and books to work with other programming languages, frameworks, APIs among other useful tools to build websites and applications.
